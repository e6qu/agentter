# Terraform Azure providers â€” `azurerm` & `azapi`

**Build model: hand-written Go over a *generated* SDK, plus a schema-driven
generic provider.** The headline is that HashiCorp **generates its own Azure SDK**
(`go-azure-sdk`, via `pandora`) rather than using Microsoft's `azure-sdk-for-go`.

Full research with all citations: [`../iac/terraform-azure.md`](../iac/terraform-azure.md).

## `terraform-provider-azurerm` â€” hand-written over a generated SDK
Resources are hand-authored Go under `internal/services/<service>/` (Plugin SDK v2
+ Plugin Framework, muxed), but they call into **`hashicorp/go-azure-sdk`**. So the
provider itself is hand-written; the SDK beneath it is generated.

## `go-azure-sdk` generated by `pandora` â€” the codegen story

feeds Microsoft's SDKs, AutoRest/TypeSpec, and Bicep):

- `importer-rest-api-specs` parses the swagger (wired as a git submodule) â†’
- `data-api` serves an intermediate model
  ([`api-definitions/`](https://github.com/hashicorp/pandora/tree/dca5fb0f724ef92619919810f0e0a216a57d2e66/api-definitions)) â†’
- [`generator-go-sdk`](https://github.com/hashicorp/pandora/tree/dca5fb0f724ef92619919810f0e0a216a57d2e66/tools/generator-go-sdk/internal/generator)

## `terraform-provider-azapi` â€” schema-driven
[`Azure/terraform-provider-azapi`](https://github.com/Azure/terraform-provider-azapi/tree/efeb3ca3c0dee7d75a1ebde92ae58a7be587715b/internal/azure/generated)

[`scripts/bicep-types-update.sh`](https://github.com/Azure/terraform-provider-azapi/blob/efeb3ca3c0dee7d75a1ebde92ae58a7be587715b/scripts/bicep-types-update.sh).

## The shared root
In Azure, **`azure-rest-api-specs` is the common source of four generated
artifacts**: `azure-sdk-for-*` (AutoRest/TypeSpec), `bicep-types-az` (Bicep +
azapi), and `go-azure-sdk` (pandora â†’ azurerm). All three HashiCorp repos
(`azurerm`, `go-azure-sdk`, `pandora`) and `azapi` are **MPL-2.0**.

| Layer | Build model |
|---|---|
| `terraform-provider-azurerm` | hand-written Go (SDKv2 + Framework) |
| `go-azure-sdk` (underneath azurerm) | **generated by pandora** from azure-rest-api-specs |
| `terraform-provider-azapi` | **schema-driven** (embeds generated Azure types) |
