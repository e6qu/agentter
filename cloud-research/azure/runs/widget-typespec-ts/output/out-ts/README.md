# WidgetService client library for JavaScript

Minimal TypeSpec service to drive the Azure TypeScript emitter.

Key links:

- [Package (NPM)](https://www.npmjs.com/package/WidgetService)

## Getting started

### Currently supported environments

- LTS versions of Node.js

### Install the `WidgetService` package

Install the WidgetService client library for JavaScript with `npm`:

```bash
npm install WidgetService
```
