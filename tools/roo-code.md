# Roo Code

**Type**: Open Source VS Code Extension  
**License**: Open Source  
**Primary Developer**: Community (forked from Cline)  
**Repository**: https://github.com/RooVetGit/Roo-Code  

## Overview

Roo Code is a VS Code extension built on Cline's foundation with enhanced multi-agent capabilities and specialized modes.

Source: [Roo Code GitHub Repository](https://github.com/RooVetGit/Roo-Code)

## Key Features

### Multi-Agent in VS Code

Roo Code supports multiple specialized agents:

> "Multi-agent orchestration within VS Code"

Unlike Cline, Roo Code can coordinate multiple agents with different specializations.

Source: [Roo Code Comparison](https://sanj.dev/post/comparing-ai-cli-coding-assistants)

### Specialized Modes

Built-in modes for different tasks:

| Mode | Purpose |
|------|---------|
| Ask | Questions and explanations |
| Architect | System design and planning |
| Code | Implementation |
| Debug | Troubleshooting |
| Orchestrator | Coordinate multiple tasks |
| Custom | User-defined modes |

Source: [Roo Code Modes](https://sanj.dev/post/comparing-ai-cli-coding-assistants)

### Auto-Approve Options

Configure automatic approval for trusted operations:

> "Auto-approve options for trusted operations"

Can auto-approve:
- File reads
- Specific file types
- Certain commands

Source: [Roo Code Auto-Approve](https://sanj.dev/post/comparing-ai-cli-coding-assistants)

### Custom Mode Creation

Define custom modes:

> "Custom mode creation"

Create modes tailored to specific project needs or team workflows.

Source: [Roo Code Custom Modes](https://sanj.dev/post/comparing-ai-cli-coding-assistants)

### MCP Support

Full Model Context Protocol integration:

> "Full MCP support"

Source: [Roo Code MCP](https://sanj.dev/post/comparing-ai-cli-coding-assistants)

## Installation

Install from VS Code marketplace:

```
Search: "Roo Code" in VS Code Extensions
Or: https://marketplace.visualstudio.com/items?itemName=RooVeterinaryInc.roo-cline
```

Source: [Roo Code Installation](https://github.com/RooVetGit/Roo-Code)

## Supported Models

Roo Code supports multiple providers:

- Claude
- GPT
- Gemini
- Local models via various providers

Source: [Roo Code Models](https://github.com/RooVetGit/Roo-Code)

## Pricing

- **Base Cost**: Free extension
- **Model Cost**: BYOK - pay your chosen provider
- **Free Tier**: Unlimited

## Limitations

According to reviews:

- VS Code only
- Can be resource-intensive
- Learning curve for mode system
- Limited context compared to CLI tools

Source: [Roo Code Limitations](https://sanj.dev/post/comparing-ai-cli-coding-assistants)

## Relationship to Cline

Roo Code is a fork of Cline with additional features:
- Multi-agent support (not in Cline)
- More specialized modes
- Enhanced auto-approve options

Source: [Roo Code GitHub](https://github.com/RooVetGit/Roo-Code)
